# CS 300: Data Structures and Algorithms

## Overview

This repository contains projects completed for CS 300: Data Structures and Algorithms at Southern New Hampshire University. The work focuses on analyzing algorithm performance, comparing data structures, and developing efficient solutions for storing, sorting, and retrieving information using C++.

Artifacts included in this repository demonstrate the evaluation and implementation of vectors, hash tables, and binary search trees, as well as the development of a course advising application capable of loading course data, displaying courses in alphanumeric order, and retrieving prerequisite information for individual courses.

## Reflection

The primary objective of these projects was to determine how different data structures affect the efficiency of storing, searching, and organizing data. Through runtime and memory analysis, I evaluated vectors, hash tables, and binary search trees to identify the most effective structure for managing course information within an academic advising system. This process reinforced the importance of selecting the right data structure for a given problem rather than focusing solely on implementing functionality.

Throughout the course, I encountered challenges related to algorithm implementation, file processing, debugging, and validating program output. Overcoming these issues required breaking larger problems into smaller components, testing incrementally, and carefully analyzing program behavior. As the projects progressed, I developed a stronger appreciation for planning software before coding, including evaluating performance tradeoffs and designing modular solutions. This experience improved my ability to create programs that are maintainable, readable, and adaptable by emphasizing clear organization, single-purpose functions, consistent coding practices, and thoughtful software design.
